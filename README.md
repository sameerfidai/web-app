# Portfolio Website

**Live Site:** [sameerfidai.com](https://www.sameerfidai.com/)

Personal portfolio showcasing my software engineering projects and experience. Built with Next.js 14, TypeScript, and Tailwind CSS.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS

## Local Development
```bash
git clone https://github.com/sameerfidai/web-app.git
cd web-app
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view locally.
